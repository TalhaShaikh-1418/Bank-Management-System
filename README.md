# 🏦 Bank Management System

Console-based Bank Management System using C++

## 🔧 Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

## 📌 Features
- Create New Account
- Display All Accounts
- Deposit Money
- Withdraw Money
- Delete Account

## 📁 Data Storage
- All records are stored in `accounts.txt` using file handling.
